 managing the learning rate?

  - It scales the learning rate differently for different coefficients. This can be useful when some features rarely take
    non-zero values.


What good does the bucketing/binning do?

    - It takes a variable and breaks it down into categories that can be true or false and stores the number of items
    that are true or false within those categories. For example: Population can be broken down into population between
    0 and 5000, 5000 and 25000, and all population greater than 25000. We can then bucket and count how many items fall
    in each bucket or category which reduces the data stored.
